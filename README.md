## Installation
### Go (1.15+)

```shell
$ go install github.com/farukak/terrathor@latest
```

## CLI Usage

```shell
$ ./terrathor.exe provider
```

Create State File

```shell
$ ./terrathor terraform
```

Create VPC

```shell
$ ./terrathor vpc
```

Create EC2

```shell
$ ./terrathor ec2
```


## Terraform Usage

Initialize Terraform

```shell
$ terraform init
```

Show terraform plans(optional)

```shell
$ terraform plan
```

Apply terraform plans

```shell
$ terraform apply
```
